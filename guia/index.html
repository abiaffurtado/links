Agente de Comunicação LigueLead

Claude + SMS + Voz + SMS Flash · n8n · Make · Webhooks

Configure em 4 passos

Transforma o Claude em um executor de comunicação.
Passa a estratégia ou arquivo — ele executa, automatiza e retorna status.

💬 SMS 📞 Voz automatizada ⚡ SMS Flash ⚙️ n8n / Make 🔔 Webhook de status

⚡ Cole suas credenciais aqui — o comando é gerado automaticamente

Os comandos abaixo são atualizados em tempo real com seus dados.

🔑 Obter credenciais em areadocliente.liguelead.app.br → Integrações → API Token →
1
Pegue suas credenciais
API Token + App ID da LigueLead

Acesse a área do cliente, crie um App e cole os valores no campo acima:

🔑
areadocliente.liguelead.app.br
Integrações → API Token → Criar App
→
2
Instale o MCP da LigueLead
Rode no terminal — fora do Claude

Rode este comando no terminal (suas credenciais já estão preenchidas):

claude mcp add -s user liguelead -e LIGUELEAD_API_TOKEN=SEU_TOKEN -e LIGUELEAD_APP_ID=SEU_APP_ID -e TRANSPORT=stdio -- npx -y @liguelead/mcp-server
ℹ️ Deve aparecer: "Added stdio MCP server liguelead". Se já existir, remova primeiro: claude mcp remove liguelead -s user
3
Crie a pasta e baixe o agente
Um único comando faz tudo

No terminal, rode:

mkdir -p agente-liguelead && cd agente-liguelead && curl -fsSL https://raw.githubusercontent.com/liguelead/agente-comunicacao/main/CLAUDE.md -o CLAUDE.md && claude
4
Passe a estratégia e execute
O agente resume, confirma e envia — ou monta o fluxo visual

Você pode passar a comunicação de 3 formas:

💬
Instrução direta
"Liga para esses números cobrando a fatura: 11999..."
📋
Régua descritiva
"D+0 SMS boas-vindas, D+3 oferta, D+7 ligação"
📄
Arquivo .md
Cole o caminho de um arquivo com etapas e lista de leads
⚙️
Fluxo visual
"Cria esse fluxo no n8n" ou "monta no Make"

O agente faz um resumo → você confirma → ele executa via LigueLead e/ou cria o workflow.

Formatos de entrada

📄
Arquivo .md com lógica + leads
Crie uma vez, reutilize sempre — o agente lê e executa

Crie um arquivo .md com as etapas da régua e a lista de contatos. O agente lê o arquivo, extrai tudo e confirma antes de enviar.

# Régua de Retenção ## Etapas | Etapa | Timing | Canal | Mensagem | |-------|--------|----------|---------------------------------| | 1 | D+0 | SMS | "Olá {nome}, sentimos sua falta!"| | 2 | D+3 | Voz | audio_id: abc123 | | 3 | D+7 | Flash | "⚠️ Oferta exclusiva expira hoje"| ## Leads | Nome | Telefone | Variáveis | |-------------|-----------------|-------------------| | João Silva | +5511999990001 | vencimento=10/06 | | Maria Souza | +5511999990002 | vencimento=12/06 |
ℹ️ Diga ao agente: "Executa o arquivo regua-retencao.md" — ele lê, resume e aguarda confirmação.
⚙️
Fluxo visual no n8n ou Make
O agente monta e importa o workflow automaticamente

Peça ao agente para criar o fluxo na plataforma que você já usa. Ele vai pedir a URL + API key e montar tudo:

🔄
n8n
Cria workflow com trigger → nós de envio → condicional de canal → wait entre etapas. Importa via API.
🔧
Make
Monta cenário com módulos HTTP → Router de canal → Sleep. Importa direto na sua organização.
ℹ️ Diga: "Cria esse fluxo no n8n" — o agente pergunta a URL e o token, monta e importa.
🔔
Webhook de retorno de status
2 etapas: agente monta o receptor → você cola a URL na LigueLead

O webhook funciona em duas etapas simples:

⚙️
1. Agente monta o receptor
Cria o nó webhook no n8n ou Make e te entrega a URL do endpoint.
🔑
2. Você cola na LigueLead
Área do cliente → Integrações → campo Webhook. Cola a URL e salva.
✅
SMS entregue / falhou
Notificação imediata de entrega ou falha de cada mensagem.
📞
Ligação atendida / não atendida
Sabe em tempo real quem atendeu e quem ignorou a chamada.
ℹ️ Diga: "Monta o receptor de webhook no n8n" — o agente cria o nó e te dá a URL para colar na LigueLead.
📊
Gerar relatório de performance em PDF
O agente gera automaticamente ao encerrar uma campanha

O agente gera um PDF de performance com design profissional sempre que você pedir "relatório", "PDF" ou ao encerrar uma régua completa.

📋
Resumo executivo
KPIs principais: total enviados, entregues, taxa de entrega, ligações atendidas e taxa de atendimento.
📅
Performance por dia e canal
Tabela diária com SMS, SMS Flash e Voz separados — enviados, entregues, falhas e taxa.
📊
Detalhe por canal
Consolidado total de SMS, Flash e Voz com todas as métricas de entrega.
💡
Observações automáticas
Análise da performance e recomendações para a próxima campanha — geradas pelo agente.

Para gerar, diga ao agente: "Gera o relatório da campanha" — ele preenche os dados e entrega o PDF.

ℹ️ Requer pip install reportlab na máquina onde o agente está instalado.
