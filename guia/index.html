Agente de Comunicação LigueLead

Guia de instalação — Claude + SMS + Voz + SMS Flash

Configure em 4 passos

Transforma o Claude em um agente especialista em comunicação automatizada.
Cria réguas personalizadas e envia mensagens diretamente via LigueLead.

💬 SMS 📞 Voz automatizada ⚡ SMS Flash

⚡ Cole suas credenciais aqui — o comando é gerado automaticamente

Os comandos abaixo são atualizados em tempo real com seus dados.

🔑 Obter credenciais em areadocliente.liguelead.app.br → Integrações → API Token →
1
Pegue suas credenciais
API Token + App ID da LigueLead

Acesse a área do cliente, crie um App e cole os valores no campo acima:

🔑
areadocliente.liguelead.app.br
Integrações → API Token → Criar App
→
2
Instale o MCP da LigueLead
Rode no terminal — fora do Claude

Rode este comando no terminal (suas credenciais já estão preenchidas):

claude mcp add -s user liguelead -e LIGUELEAD_API_TOKEN=SEU_TOKEN -e LIGUELEAD_APP_ID=SEU_APP_ID -e TRANSPORT=stdio -- npx -y @liguelead/mcp-server
ℹ️ Deve aparecer: "Added stdio MCP server liguelead". Se já existir, remova primeiro: claude mcp remove liguelead -s user
3
Crie a pasta e baixe o agente
Um único comando faz tudo

No terminal, rode:

mkdir -p agente-liguelead && cd agente-liguelead && curl -fsSL https://raw.githubusercontent.com/liguelead/agente-comunicacao/main/CLAUDE.md -o CLAUDE.md && claude
4
Descreva sua dor
O agente cria a régua e pode enviar na hora

Na primeira conversa, o agente coleta contexto do seu negócio e integrações existentes. Depois é só descrever o problema:

"Clientes somem após a primeira sessão. Cria uma régua de retenção." "Tenho 20% de inadimplência. Quero recuperar sem perder o cliente." "Agenda visitas mas metade não aparece. Reduz as faltas."
